String : 가장 많이 다루는 데이터 타입

  • java에서의 String method와 거의 유사
  • 문자열 데이터는 index(0부터 시작)를 가지고 있다.
  • 문자열 값 내에서 'br' 개행(줄바꿈) 하는 것은 추천하지 않는다.
  • new String() : 객체 타입으로 선언 가능하나, 추천하지 않는다.
  • length : 문자열의 총 길이
  • 추출 관련 메서드 : substring(), substr(), slice()
  • 문자대체관련 메서드 : replace(old, new) : 처음 찾은 값만 가능
  • trim() : 양쪽 공백 제거
  • padStart(길이, 채울문자) / padEnd(길이, 채울문자) ex. 010101-4******* / ***** 시 ...
  • charAt(index) / indexOf() / lastindexOf() ...
  • startWith("문자열") / endWith("문자열") : 해당 문자열 앞 / 뒤에 문자열이 존재하는지 확인하는 메서드 > return 값은 true / false
