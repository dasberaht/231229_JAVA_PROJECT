이벤트 객체는 전역객체로써 모든 태그에 객체를 참조시킬 수 있다.

  • a tag / button tag에 이벤트를 주로 참조시켜 사용 일반적으로 '클릭에 대한 반응을 기대할 수 있게 만듦'
  • *a tag는 href속성에 의해 자체이동(동기화)방식이 구현되어 있어, 별도의 이동을 막는 코드가 필요
  • 객체에 이벤트를 부여하는 방식이 크게 3가지(각각 목적과 작동방식이 다름)
  • 화면의 출력 영역을 document영역(객체로 인식) 라고 한다.('*DOM영역')
  • tag는 html 'element', 속성 'attribute', 일반텍스트 'textNode'라고 한다.
  • html로 출력하기 : innerHTML property 사용
  • text로 출력하기 : innerText property 사용
  • 1) html element를 가져올 때, (일반적으로)id값으로 가져오는 방법 getElementByID('idName')
  • 2) html element를 가져올 때, class로 가져오는 방법 querySelector('selector') => #, . 선택자 가능
