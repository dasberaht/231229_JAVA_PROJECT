location 객체는 브라우저의 주소표시줄 입력에 관련된 객체

  • window.location // window 생략가능
  • location.href = 'url'; > 해당 url로 이동, history 객체에 저장
  • location.replace('url'); > 해당 url로 이동하나, history 객체에 저장되지 않는다.(뒤로가기/앞으로가기를 사용할 수 없다. ex. 결제창 등)
  • location.assign('url'); > 해당 url로 이동, history 객체에 저장(href와 동일)
  • location.reload(); > 현재 페이지 새로고침
