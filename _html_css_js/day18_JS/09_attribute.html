element 속성은 크게 두 가지로 구분 1) 태그 속성 2) css 속성(style)

  • 태그의 속성은 속성명을 그대로 사용(ex. tagName.id, tagName.src, tagName.href ...)
  • class 처럼 여러 개의 값을 갖는 속성들은 별도의 속성명을 사용(ex. tagName.classLIst ...)
  • css 관련 속성은 style. 을 통해 속성(프로퍼티)에 접근 가능(ex. tagName.style.color ...)
  • css 속성 중 font-size > -(하이픈) 과 띄어쓰기를 지우고 첫 글자를 대문자로 변경하여 사용 (ex. fontSize = ""; 카멜케이스)
