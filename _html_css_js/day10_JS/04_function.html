함수(function) : 특정 기능을 수행하기 위해 작성한 코드 - 호출되어야만 실행 - 자바에서의 메서드처럼 호출하여 실행 - 자바에서는 class > method 생성하지만, - *자바스크립트에서는 어디서든 function 생성 가능

  • function 이라는 키워드를 사용하여 정의
  • function '함수명'('파라미터(매개변수; 없으면 생략)'){ 로직 작성; return 값; }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 작성(없어도 구문에 ( )는 필수로 작성)
  • 자바스크립트에서는 `파라미터의 타입 자유도가 높아`서 다양한 형태로 사용가능
  • return; 함수종료 역할과 동시에 특정 결과값을 호출한 위치로 반환
