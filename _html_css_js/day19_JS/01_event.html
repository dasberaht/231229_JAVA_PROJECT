document 영역에서 사용되는 이벤트를 정의

  • 마우스 이벤트 : click(), focus(), input(), change(), mouseover(), mouseout() (구문 > click을 작성 위치에 바꿔 사용)
  • 키보드 이벤트 : keyDown(), keyPress(), keyUp()
  • 기타 이벤트 : preventDefault() > 이전에 있는 이벤트 default 처리, target, submit()
  • addEventListener('eventname', callback fucntion); 메서드를 사용하여 구현하는 것을 권장 이벤트 중복 부여 가능
